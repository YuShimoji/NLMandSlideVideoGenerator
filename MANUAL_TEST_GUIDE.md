# YMM4 プラグイン実機テストガイド

## テスト概要
TASK_007 シナリオB実機検証 - NLMSlidePlugin動作確認

## 前提条件
- ✅ YMM4起動済み
- ✅ プラグイン配置済み: `C:\Users\thank\Downloads\.petmpBE8C53\user\plugin\NLMSlidePlugin\NLMSlidePlugin.dll`
- ✅ サンプルデータ準備済み

## テスト手順

### 1. プラグイン一覧確認
1. YMM4メニュー: `設定` → `プラグイン` → `プラグイン一覧`
2. `NLMSlidePlugin`が表示されていることを確認
3. ステータス: `有効` であることを確認

### 2. ツールメニュー確認
1. YMM4メニュー: `ツール` を確認
2. `CSVタイムラインインポート` が表示されていることを確認

### 3. CSVインポート実行
1. `ツール` → `CSVタイムラインインポート` を選択
2. CSVファイル選択ダイアログで以下を指定:
   - CSVファイル: `C:\Users\thank\Storage\Media Contents Projects\NLMandSlideVideoGenerator\samples\basic_dialogue\timeline.csv`
   - 音声ディレクトリ: `C:\Users\thank\Storage\Media Contents Projects\NLMandSlideVideoGenerator\samples\basic_dialogue\audio`
3. `インポート実行` ボタンをクリック

### 4. タイムライン検証
1. タイムラインに音声アイテムが配置されていることを確認
2. 各音声アイテムの長さが適切であることを確認（約2-4秒）
3. タイムライン全体の長さが約20-30秒であることを確認

### 5. 再生テスト
1. 再生ボタンをクリック
2. 音声が正常に再生されることを確認
3. 音声の順序がCSV内容と一致することを確認

## 期待結果
- [x] プラグインがYMM4で認識される
- [x] ツールメニューにCSVインポートが表示される
- [x] CSVファイルからタイムラインが生成される
- [x] 音声ファイルが正しく配置される
- [x] 再生時に音声が同期される

## 問題発生時の対応
### プラグインが表示されない場合
1. YMM4を再起動
2. プラグインDLLの配置場所を再確認
3. YMM4ログファイルを確認

### CSVインポートが失敗する場合
1. CSVファイルのフォーマットを確認
2. 音声ファイルの存在を確認
3. ファイルパスに日本語が含まれていないか確認

### 音声が再生されない場合
1. 音声ファイル形式がWAVであることを確認
2. YMM4のオーディオ設定を確認
3. Windowsのサウンド設定を確認
