# YMM4 Video Templates (SP-052)

YMM4 プロジェクトテンプレートの配置先。

## ディレクトリ構成

```
video_templates/
├── default/                  # 標準テンプレート (2人対話)
│   ├── template.y4mmp        # YMM4プロジェクトテンプレート (人間が作成)
│   ├── characters/
│   │   ├── reimu/            # れいむ立ち絵セット
│   │   └── marisa/           # まりさ立ち絵セット
│   └── assets/
│       ├── lower_third.png   # ローワーサード背景
│       └── title_bg.png      # 章タイトル背景
├── news/                     # ニュース風テンプレート
└── educational/              # 教育風テンプレート
```

## テンプレート作成手順 (人間向け)

1. YMM4 で新規プロジェクト作成 (1920x1080, 30fps)
2. Layer 2: 背景ベース (単色 or グラデーション)
3. Layer 5: キャラクター立ち絵を配置
   - れいむ: 画面右下
   - まりさ: 画面左下
   - リップシンク設定を有効化
4. Layer 6: 字幕位置・フォント・アウトラインを設定
5. Layer 7: テキストオーバーレイのプレースホルダー (任意)
6. `template.y4mmp` として保存

所要時間目標: 初回30分、2回目以降は既存テンプレートの微調整で5-10分。
